# 스마트팜

1. 각종 모듈을 이용하여 아두이노로 측정

2. 블루투스 모듈을 이용하여 라즈베리파이에서 코드 업로드 및 데이터를 추출

3. 추출된 데이터를 라즈베리파이에서 PC로 와이파이를 통해 소켓통신으로 전송

4. PC에서 쌓인 DB를 기반으로 웹사이트 구현 및 딥러닝을 통해 각종 병해충 판단

### HC-06 블루투스 모듈
// 블루투스 모드 진입

sudo bluetoothctl

// 블루투스 스캔

scan on

default-agent

// 블루투스 연결

pair 블루투스주소

// 블루투스 스캔 종료

scan off

//블루투스 모듈 시리얼포드 열기 및 연결

sudo rfcomm connect hci0 주소
